# 동화책 대여 서비스 (Dhwa Project)

동화에서 D를 본따 D-Hwa로 프로젝트명을 정하였습니다.<br>
아이들을 위한 동화책을 대여할 수 있는 Flask 기반 웹 프로젝트입니다.<br>
이미지 기반 도서 등록, 장바구니, 회원가입/로그인, 관리자 기능을 포함합니다.

## 주요 기능
- 도서 목록 및 검색
- 장바구니 기능
- 상품페이지, 관리자 도서 관리 페이지 (my role)
- SQLite 기반 DB + Flask-Migrate
